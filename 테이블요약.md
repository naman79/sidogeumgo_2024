# 시도금고 회계 관리 시스템 - 테이블 요약

## 시스템 개요

이 시스템은 **지방자치단체 시도금고의 재정 관리**를 위한 종합 회계 시스템입니다.

- 공금 계좌의 세입/세출 거래 처리
- 자금 운용 계좌 관리
- 각종 회계 보고서 생성 (일계표, 월계표)
- 수납 집계 및 조정 관리
- 영업일 기준 데이터 처리

---

## 테이블 사용 패턴 요약

1. **핵심 트랜잭션**: `ACL_SIGUMGO_SLV`, `ACL_SIGUMGO_MAS`가 거의 모든 조회에서 사용
2. **보고서 생성**: `RPT_` 접두사 테이블들이 각종 보고서 데이터 관리
3. **코드 관리**: `RPT_CODE_INFO`, `SFI_CMM_C_DAT`로 코드값 관리
4. **매핑 관리**: 다양한 `_MAPP`, `_MAP` 테이블로 연관 관계 관리
5. **일자 관리**: `MAP_JOB_DATE`로 영업일 계산
6. **집계 데이터**: 일계표(DDAC_TAB), 월계표(M_GYE_TAB), 잔액(JAN) 테이블들로 집계 데이터 관리

## 테이블 명명 규칙

- **ACL_**: Access Control Layer - 핵심 트랜잭션 및 마스터 테이블
- **RPT_**: Report - 보고서 관련 테이블
- **SFI_**: System Foundation Information - 시스템 기초 정보
- **MAP_**: Mapping - 매핑 테이블
- **NIO_**: Network I/O - 네트워크 연동 관련
- **WT_**: Work Table - 작업용 임시 테이블
